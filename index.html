Using scanf to Capture Input in C!

char name[50];

printf("Enter your name: ");

scanf(" %49s", name);

The scanf function is used to capture user input. Here, we are capturing a string input into the name variable. Note the space before %49s, which helps avoid capturing newline characters from previous input.

Good Practice with scanf!

char address[100];

printf("Enter your address: ");

scanf(" %99[^\n]", address);

Using the space before the format specifier and [^\n] helps to read a line of text, including spaces, up to the newline character.

Using the rand Function to Generate Random Numbers!

#include <stdlib.h>

#include <time.h>

int randomNumber;

srand(time(0));

randomNumber = rand() % 3 + 1;

The rand function is used to generate random numbers. Here, we generate a random number between 1 and 3. The srand(time(0)) is used to seed the random number generator to ensure different results each run.

Generating Random Numbers between 0 and 2!

int randomNumber;

srand(time(0));

randomNumber = rand() % 3;

Here, we generate a random number between 0 and 2 by using rand() % 3. The modulus operator ensures the result is within the desired range.
